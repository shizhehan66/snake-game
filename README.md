# 贪吃蛇小游戏

这是一个基于 HTML/CSS/JavaScript 的贪吃蛇小游戏。你可以在浏览器中打开 `index.html` 运行游戏。

## 运行方式

- 直接打开 `index.html`
- 或使用 VS Code 的 Live Server 扩展预览

## 游戏说明

- 使用方向键 `↑ ↓ ← →` 控制蛇移动
- 吃到食物，蛇会变长，分数会增加
- 撞墙或撞到自己时游戏结束

## 推送到 GitHub

1. 在项目根目录打开终端
2. 运行：
   ```bash
   git init
   git add .
   git commit -m "初始化贪吃蛇小游戏"
   ```
3. 在 GitHub 上创建新仓库
4. 连接远程仓库：
   ```bash
   git remote add origin https://github.com/你的用户名/仓库名.git
   git branch -M main
   git push -u origin main
   ```
